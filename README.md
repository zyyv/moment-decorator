# moment-decorator

精致您的朋友圈～

## ✨ Features
- Vite 极速开发体验
- TypeScript 支持
- 文件系统路由（unplugin-vue-router + layouts）
- UnoCSS 原子化 & Attributify & 图标 (presetIcons)
- 自动按需导入（API & 组件）

## 🚀 Setup

```bash
pnpm install
pnpm dev
```
访问: http://localhost:5173

### 其他脚本
```bash
pnpm build      # 类型检查 + 打包
pnpm preview    # 预览构建产物
pnpm lint       # ESLint 检查
pnpm lint:fix   # 自动修复
```
